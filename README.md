# CorreioWs

Java SOAP Client generated for Correios WS

## Packing

```
$ mvn clean generate-sources
```

## Alternative

```
$ mvn org.apache.cxf:cxf-codegen-plugin:3.0.1:wsdl2java@generate-cxf -Dwsdl.path=<path to wsdl>
```
